\section*{Derivation of the Gaussian from the Code}
\begin{verbatim}
xg = (0.83255461 / hwhm) * wg
gpro = np.exp(-xg*xg)
\end{verbatim}


\[
    \sqrt{\ln(2)} = \sqrt{0.69314718} = 0.83255461
\]

\[
    xg = \frac{0.83255461}{hwhm} \, wg
    \qquad
    xg =\frac{\sqrt{\ln(2)}}{hwhm} \, wg
\]

\[
    gpro =    \exp(-xg^2)
    \qquad
    \exp\left(
    -
    \frac{\ln(2)}{hwhm^2}
    \, wg^2
    \right)
    \qquad
    gpro =
    \exp\left(
    -
    \ln(2)
    \left(
        \frac{wg}{hwhm}
        \right)^2
    \right)
\]

At

\[
    wg = hwhm
    \qquad
    gpro =
    \exp(-\ln(2))
    =
    \frac{1}{2}
    \qquad
    \mathrm{FWHM} = 2 \cdot hwhm.
\]


\section*{Relation Between HWHM and $\sigma$}

The standard Gaussian

\[
    G(w) = \exp\left( -\frac{w^2}{2\sigma^2} \right).
\]

Matching exponents
\[
    - \ln(2)\frac{w^2}{\mathrm{HWHM}^2}
    =
    - \frac{w^2}{2\sigma^2}.
    \qquad
    =>
    \frac{2 * ln(2)}{\mathrm{HWHM}^2} =\frac{1}{\sigma^2}
    \qquad
    \sigma = \frac{\mathrm{HWHM}}{\sqrt{2\ln(2)}}.
\]

\section*{Kernel Width in the Implementation}

\begin{verbatim}
    nhalf = int(3.3972872*hwhm/dw) # 4.0 / sqrt(2.0*numpy.log(2.0)) = 3.3972872
\end{verbatim}

In the code, the half-width in index space is
where $\Delta\lambda$ is the model wavelength spacing.

\[
    \text{nhalf}
    =
    \frac{4}{\sqrt{2\ln(2)}}
    \frac{\mathrm{HWHM}}{\Delta\lambda},
\]
Using: which tells us that Gaussian extends in wavelength space to

\[
    \sigma = \frac{\mathrm{HWHM}}{\sqrt{2\ln(2)}},
    \qquad
    \text{nhalf}
    =
    \frac{4\sigma}{\Delta\lambda}.
    \qquad
    \pm 4\sigma.
\]


\begin{verbatim}
    ng = 2 * nhalf + 1
    wg = dw * (np.arange(ng) - (ng-1)/2.0)

\end{verbatim}

In the code, the number of kernel points is defined as: \(  ng = 2\,nhalf + 1. \)

The index array is constructed as: \(  \text{np.arange}(ng) = \{0,1,2,\dots,ng-1\}. \)

Then the code shifts this array by: \(  \frac{ng-1}{2} \rightarrow  \frac{ng-1}{2} = \frac{2\,nhalf}{2} = nhalf. \)

so the Gaussian kernel extends to

\[
    nhalf \cdot dw = 4\sigma,
    \qquad
    \pm 4\sigma.
\]
