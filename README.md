# IDL to Python Simulator Port – CI Sandbox

This repository started as a minimal proof of concept to set up
- a clean Python project structure
- pytest-based unit testing
- automated CI using GitHub Actions

The initial commits contain only trivial, deterministic example code used to validate the testing and CI pipeline.

In later stages, this repository will host Python code ported from an IDL simulator developed in the context of an ESA mission. 
That code will be introduced incrementally, together with proper unit tests.
